《JavaScript高级程序设计》读书笔记（二）

发表于 2017-02-16 | 分类于 JavaScript ， 读书笔记 | | 阅读次数

5.1 Object类型

创建Object实例的两种方式：

1.new操作符后跟Object构造函数
2.对象字面量表示法

字面量表示法是较常使用的方式，也是向函数传递大量可选参数的首选方式

注：通过字面量定义对象时，实际上不会调用Object的构造函数

5.2 Array 类型

创建数组的两种方式：

1.使用Array构造函数
2.使用数组字面量表示法

数组的length不是只读的，通过这个属性可以从数组末尾移除或添加新项目

5.2.1 检测数组

使用instanceof检测数组

instanceof操作符的问题在于，它假定只有一个全局执行环境。如果网页中包含多个框架，从而存在多个不同版本的Array构造函数，如果你从一个框架向另一个框架传入一个数组，那么传入数组与在第二个框架中原生创建的数组分别具有各自的不同的构造函数。instanceof就失效了 (⊙﹏⊙)b

因此在ECMAScript5中新增了Array.isArray()方法

5.2.2 转换方法

所有对象都具有toLocaleString()、toString()、valueof()方法。

String()与toLocaleString()的区别 （高程上没提 o(╯□╰)o，感谢万能的MDN！）

本质上两个方法作用是几乎一样的，只有toLocaleString()执行时会根据当前字符环境输出对应的字符格式，这个格式也是可以以参数的形式传入的

5.2.3 栈方法

栈是一种LIFO（Last-In-First-Out，先进后出）的数据结构。ECMAScript为数组提供了push()和pop()方法，以便实现类似栈的行为。

5.2.4 队列方法

队列是一种FIFO（First-In-First-Out，先进先出）的数据结构。结合使用shift()和push()方法，实现类似队列，unshift()和pop()实现相反方向的模拟队列。

5.2.5 重排序方法

数组中有两个直接可用的排序方法 reverse() 、sort()。

sort()默认排序规则，会调用每个数组项toString()转型方法，然后比较字符串，因此 11会排在5前面,sort()可以接收一个比较函数作为参数，以便自定义比较规则

5.2.6 操作方法

slice()提取数组的一部分，并返回这个新数组

两个参数表示起点和终点

splice()方法通过删除现有元素和/或添加新元素来更改数组的内容。

三个参数表示起点、删除个数、插入内容

5.2.7 位置方法

indexOf()返回要查找的项在数组中的位置，没找到返回-1。

比较时会使用全等操作符

lastIndexOf()从数组末尾向前查找，与indexOf()相反

5.2.8 迭代方法

every() 对数组中每一项运行给定函数，如果该函数对每一项都返回true，则返回true。

some() 对数组中每一项运行给定函数，如果该函数对任意一项都返回true，则返回`true。

forEach() 对数组中每一项运行给定函数，无返回值。

filter() 对数组中每一项运行给定函数，返回该函数会返回true的项组成的数组。

map() 对数组中每一项运行给定函数，返回每次函数调用结果组成的数组

以上方法都不会修改数组中的包含的值。

5.2.9 归并方法

reduce() 接受四个参数：前一个值、当前值、项的索引、数组对象，函数返回的任何值都会作为第一个参数自动传给下一项。

reduceRight()遍历方向是从后到前。
