2017第一波我的前端面试

发表于 2017-03-18 | | 阅读次数

因为一些不可描述的原因，今年前端竞争特别激烈，据我的前端小伙伴透露的不可靠消息，当你每天早上起来每投出一份简历，就有 300 个不可描述（同上）的前端在和你竞争同一个工作，当然听到这里的时候我也是崩溃的，当我颤抖的双手投出 30+ 份简历的时候，终于有了一次面试机会…
( 注: 本人对以上消息真实性不负任何责任)

题目主要分 CSS 和 JavaScript 两部分，使劲回忆，能记起的就下面这些了…

一、CSS

1.实现 div 的垂直水平居中

1.绝对定位加负margin
2.模拟表格法
3.CSS3的transform：translate(-50%,-50%)
4.Flex布局
5.当然实现的方式不至这些，主要考察基本的布局能力，其他的可以自行 Google 了解

2.解释一下 BFC

块格式化上下文（Block Formatting Context） 是Web页面的可视CSS渲染的一部分。它是块盒子的布局发生及浮动体彼此交互的区域。 –MDN

1.BFC 有什么用呢？

  • 清除浮动
  • 避免margin重叠

2.以下方式可以创建 BFC：

  • 根元素或其它包含它的元素
  • 浮动 (元素的 float 不是 none)
  • 绝对定位的元素 (元素具有 position 为 absolute 或 fixed)
  • display为 inline-blocks、table-cell和table-caption
  • 块元素具有 overflow 值不是 visible
  • 弹性盒子 flex boxes (元素具有 display: flex 或 inline-flex)

3.说一下 position

主要就是解释下 static、relative、absolute、fixed的表示形式

4.CCS 有哪些选择器

通配符选择器、后代选择器、相邻兄弟选择器、子代选择器、

6.自定义单选框样式

用 Label 组合 radio，隐藏 radio，通过 Label 的伪类 checked切换选中状态

二、JavaScript

1.JavaScript里参数传递是按值传递还是引用

首先基本数据类型 Number String undefined Boolean Null 是按照值传递的。那 Obejct 和其他诸如数组的基本包装类型是按值还是引用呢，因为没有留意过，我只说这些传递的还是引用，所以表现还是传递的引用，回来后查阅了相关文档，原来全部都是按值传递，而引用类型的值就它的引用。

了解更多：javascript传递参数如果是object的话，是按值传递还是按引用传递？

2.接上题，那么传递一个数组，如何在不影响原数组的情况下，修改传入的数组

当时被第一题问的有点懵，不出意外的没答出来，回来想了下，可以对传入数组做一次浅拷贝，感觉还是不怎么好的方式…

3.如何创建一个对象

讲得很乱，没组织好语言，说到底还是不够熟悉…
创建构造对象的方式，《高程》上有这几种方式： 工厂模式、原型模式、组合构造函数+原型模式、动态原型模式、寄生构造函数模式、稳妥构造函数模式，其中组合构造函数+原型模式应该是最常见的一种。

4.如何实现继承

就说了下高程里提到的方式，调用父类构造函数继承熟悉，子类 prototype 指向父类的实例。这回答我都太浅了…

了解更多：再谈Javascript原型继承

5.事件冒泡的作用，以及举例不冒泡的事件有哪些

利用事件冒泡的机制，可以实现事件委托，不冒泡的事件有 mouseenter、mouseleave

了解更多：事件机制详解

6.JavaScript 如何获取 CSS3 动画的状态

真是问到我了，平时CSS3动画用的还是很多，但是通过 JavaScript 去获取动画状态还真没有做过（菜…）

查了MDN，当过渡完成时触发一个事件，在符合标准的浏览器下，这个事件是 transitionend，animate动画有对应的 animationstart 、animationend

7.JQuery 中 prop()、attr()的区别

prop是获取当前DOM元素上的JS对象属性（如checked、disabled），attr获取当前DOM元素对象的文档节点属性（如type 、name、data-*）。

8.了解 Promise 吗

Promise？ES6 真的很少用啊，急需补课！就谈了下 Promise 是异步编程的一种解决方案，大概的处理流程，初始状态Pending 变为 Fulfilled，触发 resolve()，Pending 变为 Rejected，触发 reject()，反正泛泛的说了一些…
了解更多：ECMAScript 6 入门

9.用过什么框架？

用过 Vue ，让谈下 Vue 的优点，就是双向数据绑定，避免了DOM操作，代码更简洁，学习曲线相对平缓….

10.Vue 和 jQuery 共存，通过 jQuery 改变了一个 input 的值，Vue更响应吗？

没答上，没有这样玩过，猜测了下主要是看能不能触发 setter， 回来写了个demo测试了下，确实不能的，至于为啥我还是不懂，有懂同学快来教我哇

End！
这次面试还是发现了很多知识盲区，有些知识点掌握程度不够，以至于被问到的时候有点不知道怎么表达，看来还得好好学习惹~
